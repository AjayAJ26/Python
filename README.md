# 🏨 Hotel Booking Analysis  

*A Python data analysis project focused on exploring hotel booking data, cleaning records, transforming metrics, and generating insights on occupancy, revenue, and customer behavior.*  

---

## ❗ Problem Statement  
Hotels often struggle with:  
- Understanding booking trends across properties and cities  
- Identifying overbookings and capacity issues  
- Measuring occupancy rates by room type and location  
- Tracking revenue performance and customer ratings  

Without structured analysis, it becomes difficult to optimize operations and improve decision-making.  

---

## 📝 Project Overview  
This project analyzes hotel booking datasets using **Python (Pandas, Matplotlib)** to uncover meaningful insights that support better hotel management.  

The analysis focuses on:  
1. **Data Exploration** – Understanding booking platforms, room categories, and revenue distribution  
2. **Data Cleaning** – Handling invalid guest counts, outliers, and missing values  
3. **Data Transformation** – Creating occupancy percentage metrics  
4. **Insights Generation** – Occupancy trends, revenue analysis, and customer ratings

## 🔢 Data Sources  
The datasets include:  
- 🏨 **Hotel Information** (property IDs, names, cities, types)  
- 🛏️ **Room Details** (room categories, classes, capacity)  
- 📅 **Date Dimension** (check-in dates, weekdays/weekends, months)  
- 📦 **Booking Records** (successful bookings, guests, revenue, ratings, platforms)  

---

## 📊 Analysis & Insights  

### ✅ Solution Steps  
- Imported and explored booking datasets  
- Cleaned data (removed invalid guest counts, handled outliers, filled missing values)  
- Transformed data by calculating **occupancy percentage (occ_pct)**  
- Merged datasets to generate city-level and room-level insights  

### 🌟 Key Insights  
- **Occupancy Trends**: Higher occupancy on weekends compared to weekdays  
- **City-Level Analysis**: Certain cities consistently show higher occupancy and revenue  
- **Room Categories**: Premium room classes have higher average occupancy rates  
- **Revenue Insights**: Month-by-month revenue tracking and platform contribution  
- **Customer Ratings**: Average ratings vary across cities and hotel types  

### 📌 Visualizations  
- 📊 Bar charts for booking platforms and city occupancy  
- 🥧 Pie chart for revenue contribution by booking platform  
- 📈 Revenue trends across months
