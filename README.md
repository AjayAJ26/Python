# 💰 Expense Management System  

*A Data Analytics project combining a **Streamlit frontend** and a **FastAPI backend** to manage, track, and analyze daily expenses.*  

---

## ❗ Problem Statement  

Managing personal or organizational expenses often becomes messy when data is scattered across spreadsheets or manual logs.  

Challenges include:  
- Lack of a **centralized system** for expense tracking  
- Difficulty in **updating and retrieving expenses by date**  
- Limited ability to **visualize expense categories and trends**  
- Manual workflows prone to errors and inefficiency  

To solve these issues, this project builds a **modern expense management system** with a clean UI and robust backend APIs.

---

## 📝 Project Overview  

This project provides a **Streamlit-based frontend** for user interaction and a **FastAPI backend** for data storage and retrieval.  

### 🎯 Objective  
Enable users to:  
- Add, update, and delete expenses  
- Retrieve expenses by date  
- Visualize expense breakdowns by category  
- Maintain persistent records with database integration  

---

## 🚀 Solution  

Developed a **two-tier architecture**:  
- **Frontend (Streamlit)** → Interactive forms, charts, and dashboards  
- **Backend (FastAPI)** → REST APIs for CRUD operations, connected to a database  

### 🛠️ Tools & Technologies  
- Streamlit  
- FastAPI  
- Python  
- SQL (for persistence)  

---

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/expense-management-system.git
   cd expense-management-system
   ```
2 **Install dependencies:**:   
   ```commandline
    pip install -r requirements.txt
   ```
3 **Run the FastAPI server:**:   
   ```commandline
    uvicorn server.server:app --reload
   ```
4 **Run the Streamlit app:**:   
   ```commandline
    streamlit run frontend/app.py
   ```
