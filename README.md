# 🏥 Hospital Doc-to-Data System  

*A Healthcare Data Management project combining a **Streamlit frontend**, **FastAPI backend**, and **MySQL database** to digitize patient and prescription records from hospital documents.*  

---

## ❗ Problem Statement  

Healthcare providers often struggle with **manual data entry** and **unstructured records**.  
Challenges include:  
- Time-consuming manual entry  
- Risk of human errors  
- Difficulty in managing patient & prescription records  
- Lack of centralized data repository  
- Delays in retrieving/updating patient information  

This project automates **document-to-data workflows**, enabling structured storage and analysis of healthcare records.  

---

## 📝 Project Overview  

This project demonstrates how a **data analyst** can design and implement a full-stack system to:  
- Parse hospital documents into structured datasets  
- Store patient & prescription data in a relational database  
- Provide a frontend for CRUD operations  
- Enable downstream analytics and reporting  

### 🎯 Objective  

Empower healthcare staff to:  
- Upload hospital documents  
- Automatically extract patient & prescription information  
- Manage records (add, update, delete)  
- Maintain a centralized healthcare database  

---

## 🚀 Solution  

Developed a **data-driven healthcare application** with automated document parsing and structured storage.  

### Architecture  

- **Frontend (Streamlit)** → Interactive forms & dashboards  
- **Backend (FastAPI)** → REST APIs & business logic  
- **Database (MySQL)** → Persistent storage of patient & prescription records  
- **Document Processing Module** → Extracts structured information from uploaded docs

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/expense-management-system.git
   cd expense-management-system
   ```
2 **Install dependencies:**:   
   ```commandline
    pip install -r requirements.txt
   ```
3 **Run the FastAPI server:**:   
   ```commandline
    uvicorn main:app --reload
   ```
4 **Run the Streamlit app:**:   
   ```commandline
    streamlit run app.py
   ```
